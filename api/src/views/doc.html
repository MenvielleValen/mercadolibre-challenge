Doc API de Mercado Libre Challenge

(*) firma: parámetro author enviado en headers.

Ruta Método Req. Firma Descripción Parámetros Respuesta
/api/items?q={query} GET Si Permite realizar una búsqueda de items en Mercado Libre. query (String): Consulta de búsqueda.
  • 200 OK: Retorna una lista de items encontrados.
  • 400 Bad Request: Si la consulta de búsqueda es inválida.
  • 403 Forbidden: Si no se envió la firma o es inválida.
  • 500 Internal Server Error: Si ocurre un error interno en el servidor.
/api/items/:id GET Si Permite obtener los detalles de un item específico en Mercado Libre. id (String): ID único del item.
  • 200 OK: Retorna los detalles del item solicitado.
  • 404 Not Found: Si el item no fue encontrado.
  • 403 Forbidden: Si no se envió la firma o es inválida.
  • 500 Internal Server Error: Si ocurre un error interno en el servidor.
